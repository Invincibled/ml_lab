# 正则化
正则化，是一种在训练机器学习模型时，在损失函数中添加额外项，来惩罚过大的参数，进而限制模型复杂度、避免过拟合，提高模型的泛化能力的技术。

常见的正则化技术有L1正则化，和L2正则化。  

## L1 正则化
L1正则化，也称为Lasso正则化，通过在损失函数中添加参数绝对值的和来实现。
$$
Loss = 原Loss + \alpha \sum_{i=1}^{n} |w_i|
$$
其中，$\alpha$是正则化参数，$w_i$是模型参数。


## L2 正则化
L2正则化，也称为Ridge正则化，通过在损失函数中添加参数平方和来实现。
$$
Loss = 原Loss + \alpha \sum_{i=1}^{n} w_i^2
$$
其中，$\alpha$是正则化参数，$w_i$是模型参数。  

L2 正则化通过惩罚模型参数的平方，使得所有参数都变的更小，但不会将参数强行压缩为0.它会使得模型尽量平滑，从而防止过拟合。  
在解决回归问题时，使用L2正则化也被称为“岭回归”

# ElasticNet 正则化（弹性网络回归）
ElasticNet正则化，也称为弹性网络回归，通过在损失函数中添加参数绝对值的和和平方和的加权和来实现。
$$
Loss = 原Loss + \alpha \sum_{i=1}^{n} |w_i| + \beta \sum_{i=1}^{n} w_i^2
$$
其中，$\alpha$和$\beta$是正则化参数，$w_i$是模型参数。  
ElasticNet正则化通过同时惩罚参数的绝对值和平方，使得模型参数既不会过大，也不会被强行压缩为0。它结合了L1和L2正则化的优点，既可以进行特征选择，也可以防止过拟合。  
在解决回归问题时，使用ElasticNet正则化也被称为“弹性网络回归”
