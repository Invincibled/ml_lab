# 梯度下降法
梯度下降法是一种用于最小化目标函数的迭代优化算法，核心是沿着目标函数（如损失函数）的负梯度方向逐步调整参数，从而逼近函数的最小值。梯度方向指示了函数增长最快的方向，因此负梯度方向是函数下降最快的方向。
梯度下降法是一种优化算法，用于最小化损失函数。它通过 iteratively 更新模型参数，使得损失函数不断减小，最终达到最小值。
梯度下降法的基本思想是：从一个初始点开始，沿着损失函数梯度的反方向迭代更新参数，使得损失函数不断减小，最终达到最小值。
梯度下降法的步骤如下：
1. 初始化模型参数。
2. 计算损失函数的梯度。
3. 更新模型参数。
4. 重复步骤2和步骤3，直到损失函数收敛。 
梯度下降法的公式如下：
$$
\theta := \theta - \alpha \nabla J(\theta)
$$
其中，$\theta$ 是模型参数，$\alpha$ 是学习率，$\nabla J(\theta)$ 是损失函数 $J(\theta)$ 对参数 $\theta$ 的梯度。
梯度下降法的优缺点：
优点：实现简单，计算量小，适用于大规模数据。
缺点：需要选择合适的初始点和学习率，否则可能无法收敛或收敛到局部最优解。
## 梯度下降法的变种
1. 随机梯度下降法（Stochastic Gradient Descent, SGD）  
适合场景：数据量较大，计算资源充足。
2. 批量梯度下降法（Batch Gradient Descent, BGD）
3. 小批量梯度下降法（Mini-batch Gradient Descent, MBGD）

## 梯度下降法常见问题
1. 学习率选择不当，导致无法收敛或收敛到局部最优解。
2. 初始点选择不当，导致无法收敛到全局最优解。
3. 梯度计算误差，导致无法准确更新参数。
4. 计算资源限制，导致无法进行大量迭代。

（1）特征缩放
通常需要提前对特征进行缩放（如标准化或归一化），以加快收敛速度
（2）局部极小值、鞍点问题
可能陷入局部最小值（非全局最优解），或遇到鞍点（梯度为零但非极值点）。
解决方案：使用动量（Momentum）、自适应优化器（如adam）或二阶方法（如牛顿法）


## 学习率的选择
学习率过大： 可能跳过最优解，甚至发散
学习率过小： 可能导致收敛速度过慢，容易陷入局部最优解
自适应学习率：高级优化器 如（Adam、Adagrad）动态调整学习率以提升性能
