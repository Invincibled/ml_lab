# 逻辑回归
逻辑回归是一种用于分类问题的监督学习算法，它通过学习输入特征与输出标签之间的关系，对新数据进行分类预测。逻辑回归模型假设输出标签的概率服从逻辑分布，因此得名逻辑回归。
逻辑回归的基本思想是：通过学习输入特征与输出标签之间的关系，对新数据进行分类预测。逻辑回归模型假设输出标签的概率服从逻辑分布，因此得名逻辑回归。
逻辑回归的数学公式为：
$$
h(\theta) = \frac{1}{1 + e^{-\theta^T x}}
$$
其中，$h(\theta)$ 是输出标签的概率，$\theta$ 是模型参数，$x$ 是输入特征。

逻辑回归的实现步骤如下：
1. 数据预处理：对输入数据进行预处理，包括缺失值处理、特征缩放等。

2. 特征选择：选择对输出标签有重要影响的特征，可以提高模型的泛化能力。

3. 模型训练：使用训练数据对模型进行训练，学习输入特征与输出标签之间的关系。

4. 模型评估：使用测试数据对模型进行评估，计算模型的准确率、精确率、召回率等指标。

逻辑回归通过将线性回归的输出映射到[0,1]区间，来表示某个类别的概率。

常用的映射函数有sigmoid函数和tanh函数。

逻辑回归的应用场景:
* 信用评分： 预测客户是否会违约
* 欺诈检测：预测某笔交易是否是欺诈行为
* 垃圾邮件检测：预测邮件是否是垃圾邮件
* 广告点击率预测：预测用户是否会点击广告
* 图像分类：预测图像属于哪个类别
* 情感分析：将文本分类为正面、负面或中性情感
* 产品质量分类：预测产品是否符合质量标准
* 医学诊断：预测患者是否患有某种疾病
* 蛋白质功能检测：基于蛋白质序列和结构特征预测其功能类别。

## 逻辑回归的损失函数
逻辑回归的损失函数也称为对数损失函数或交叉熵损失函数。也称为二元交叉熵损失函数。
用于衡量模型输出的概率分布与真实标签之间的差距，逻辑回归的损失函数来源于最大似然函数。
逻辑回归的损失函数为：
$$
J(\theta) = -\frac{1}{m} \sum_{i=1}^{m} [y^{(i)} \log(h(\theta^{(i)})) + (1 - y^{(i)}) \log(1 - h(\theta^{(i)}))]
$$

## API使用
scikit-learn中的LogisticRegression类用于实现逻辑回归，它支持二分类和多分类任务，并提供了多种优化算法和正则化选项。
```python
#solver:优化算法
# lbfgs:拟牛顿法(默认),仅支持L2正则化
# newton-cg:牛顿法,仅支持L2正则化
# liblinear:坐标下降法,适用于小数据集,支持L1和L2正则化
# sag:随机平均梯度下降,适用于大规模数据集,仅支持L2正则化化
# saga:改进的随机梯度下降,适用于大规模数据,支持L1、L2和ElasticNet正则化
#penalty:正则化类型,可选11、12和elasticnet
#C:正则化强度,C越小,正则化强度越大
#class_weight:类别权重,balanced表示自动平衡类别权重,让模型在训练时更关注少数类,从而减少类别不平衡带来的偏差
model = sklearn.linear_model.LogisticRegression(solver="lbfgs",
penalty="12",C=1,class_weight="balanced")
```
